# Spotify Music Chart Trend Analysis across Asian Countries using Spark and Hive
# Spark와 Hive를 활용한 아시아 국가별 Spotify 음악 차트 트렌드 분석

## 1. 문제 정의

본 프로젝트는 Spotify Chart 데이터를 활용하여 아시아 여러 국가의 음악 소비 트렌드를 비교·분석하는 것을 목표로 한다. 최근 K-pop을 포함한 다양한 음악 장르가 아시아 국가들에서 빠르게 확산되고 있으며, 국가별로 인기 있는 음악 장르와 아티스트, 차트 유지 기간, K-pop 선호도에는 차이가 존재한다. 따라서 본 프로젝트에서는 Spotify 차트 데이터를 기반으로 국가별 음악 트렌드의 차이와 시간에 따른 변화를 분석하고자 한다.

분석 대상 국가는 베트남, 한국, 일본, 대만, 홍콩, 싱가포르, 태국, 인도네시아, 필리핀, 말레이시아 등 아시아 국가로 설정한다. Global Chart는 분석 범위에서 제외하고, 아시아 국가 간 비교에 집중한다.

사용할 데이터는 Spotify Chart 또는 공개 데이터셋에서 수집한 국가별 음악 차트 데이터이다. 주요 데이터 항목은 날짜, 국가, 차트 종류, 순위, 곡명, 아티스트명, 스트리밍 수 등이다. 필요한 경우 아티스트 장르나 K-pop 여부를 구분하기 위한 추가 메타데이터를 함께 수집하거나 별도의 매핑 테이블을 생성한다.

본 프로젝트에서 해결하고자 하는 주요 분석 질문은 다음과 같다.

1. 각 아시아 국가별 Spotify 차트에서 가장 많이 등장하는 음악 장르 또는 아티스트 그룹은 무엇인가?
2. 각 국가별 Spotify 차트에서 가장 오랫동안 순위를 유지한 곡 또는 아티스트는 누구인가?
3. 아시아 국가들 중 K-pop 진입 비중이 가장 높은 국가는 어디이며, 시간에 따라 그 추이는 어떻게 변화하는가?

이 프로젝트는 여러 국가와 기간에 걸친 대량의 음악 차트 데이터를 수집하고, HDFS에 저장한 뒤 Spark와 Hive를 활용하여 전처리, 집계, 분석을 수행한다는 점에서 빅데이터 처리 시스템 설계 및 구현이라는 과제의 목적에 부합한다.

## 2. 기술 스택

본 프로젝트에서 사용할 예정인 기술 스택은 다음과 같다.

### 데이터 수집
- Python
- requests 또는 BeautifulSoup
- 공개 Spotify Chart 데이터 또는 Kaggle 공개 데이터셋

### 데이터 저장
- HDFS
- CSV 또는 Parquet 형식

### 데이터 처리 및 전처리
- Apache Spark
- PySpark

### 데이터 분석
- Apache Hive
- HiveQL
- Spark SQL

### 시각화
- Python
- Matplotlib 또는 Seaborn

Spark는 대용량 차트 데이터를 전처리하고 집계하는 핵심 도구로 사용한다. Hive는 정제된 데이터를 테이블 형태로 관리하고, 국가별·기간별·장르별 분석 쿼리를 수행하기 위해 사용한다. Python 시각화 도구는 Spark 또는 Hive에서 추출한 최종 분석 결과를 그래프로 표현하는 보조 도구로 사용한다.

## 3. 구현 계획
